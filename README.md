# borang-tempahan-mua
Borang tempahan MUA yang dibuat khas untuk memudahkan pihak MUA menguruskan tempahan-tempahan yang dibuat pelanggan.
Web form ini akan mengambil tempahan dari pelanggan.
Data yang diterima akan dicatat di Google Sheets.
Data tersebut kemudiannya akan dimasukkan ke dalam Google Calendar. (notifikasi satu hari sebelum sepatutnya, tapi masih belum fix, dia auto set 30min sebelum)
Ini akan memudahkan MUA tanpa perlu ingat setiap maklumat pelanggannya dan tanpa perlu menulisnya secara manual dalam whatsapp.
Web form ini masih boleh dicustomise, kadang-kadang kita tak sedar apa yang possible.
Jika MUA mahu menambah atau mengurangkan apa-apa feature yang sudah ada, maka dibolehkan.
Penerangan di bawah akan menerangkan tentang bahagian technical web form ini.
